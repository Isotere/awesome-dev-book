# Поток запросов

![](./images/flow_01.png)
1. Юзеры обращаются к сайту по доменному имени. 
2. ИП адрес возвращается браузеру или мобилке
3. Напрямую отправляются HTTP запросы по этому ИП адресу (веб-сервер)
4. Веб-сервер возвращает ответ (HTML, Json и тд)
